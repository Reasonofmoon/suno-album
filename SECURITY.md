# Security Policy for Reason_Moon_Album

## 지원하는 버전 (Supported Versions)
이 프로젝트는 최신 릴리스 및 AI 모델 연동의 안정성을 우선합니다.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## 취약점 보고 방법 (Reporting a Vulnerability)
보안 취약점을 발견하신 경우 아래 절차를 따라주세요:
1. **이메일 보고**: `reasonofmoon@example.com` (또는 프로젝트 관리자 이메일)로 세부 내용을 보내주세요.
2. **응답 시간**: 보고 후 48시간 이내에 확인 메일을 보내드리며, 수정 계획을 공유합니다.
3. **공개 유예**: 취약점이 해결되어 패치가 배포될 때까지는 대중에게 공개하지 말아주시길 부탁드립니다.
